# 一只收纳盒（ALittleBox）
“一只收纳盒”小程序代码
这是隐藏在岁月深处的关于音乐、情感和色彩的梦想，因为有了这一次细小的尝试，才有了以后回忆中的温暖、想起后的微笑、初见时的惊喜，我希望收藏每一个春秋、冬夏，希望收藏每一次汗水后的努力，跋涉后的获得，在这里，都能实现。
